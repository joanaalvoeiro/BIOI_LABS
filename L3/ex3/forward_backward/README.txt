The program forward_backward.py runs the Forward-Backward algorithm on a nucleotide sequence input by the user.
It then computes Pi^ for a user input index of this sequence.
It outputs the the (states x sequence) matrix generated by the Forward algorithm, then the value for P(S) computed via the Forward algorithm,
then (states x sequence) matrix generated by the Backward algorithm, and finally the value for P(S) computed via the Backward algorithm.
These two values for P(S) are exactly the same.
Finally, the program outputs P(Pi(index) = k | x), and Pi^(index).
Our program is dependent on the numpy library. To install numpy, please use the command 'pip install numpy'.
Our program is also dependent on the math library, which should come preinstalled with python.

To run:
- Open the terminal in the directory containing ex3.py
- Insert the command 'python3 ex3.py'
- You will then be prompted to insert the sequence to evaluate, please input a sequence composed of nucleotides A, T, C and G.
- You will then be prompted to insert an index, please input an integer from 1 to the total length of the input sequence.

An example of interaction with our program would be:
($ indicates the command prompt, > indicates user input)

$ python3 forward_backward/forward_backward.py
